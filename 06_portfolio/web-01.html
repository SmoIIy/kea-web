Grundlæggende web_01

Studiestartsprøve

Første rigtige emne, var intro til webudvikling og basic designprincipper. Dette var ting som gestaltlove, farver og former. Dog var dette primært et intro til web - det vil sige HTML og CSS. Der var fokus på et responsivt design, med en “mobile first” tilgang. Dette blev opnået ved hjælp af primært CSS grid, flex og media queries. Vi blev også introduceret til VS Code, browserens developer tools, og FileZilla til upload af filer på vores eget domæne.

Link

Proces

Processen var simpel at gå til, i det af layout og opsætning var bestemt af de layoutdiagrammer og wireframes som vi fik udleveret, og som skulle følges. Så den primære opgave, var at bruge de principper vi havde lært inden for HTML og CSS. Grundet min erfaring fra tidligere arbejde, var det kun CSS grid, jeg ikke havde arbejdet med før.

Den kreative proces gik med at vælge farver der passede sammen, samt små touches som rundede kanter og borders
